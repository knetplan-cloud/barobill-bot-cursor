기획자의 핵심 역할

서비스의 모든 출발점은 기획자입니다. 고객의 니즈와 페인포인트를 발견하고 이를 해결하여 서비스 개선과 성장을 이끕니다.

API 기획 핵심 원칙

바로빌 API는 파트너 개발자가 구현하고 기업 사용자가 경험합니다. 기획은 기술 스펙을 비즈니스 시나리오로 번역하는 일입니다.

요청 예시 (기술적 관점)
POST /invoices
{
"type": "TAX",
"issuer": {"bizNo": "123-45-67890"},
"receiver": {"bizNo": "987-65-43210"},
"items": [{"name": "상품A", "qty": 10}]
}
응답 예시 (기술적 관점)
200 OK
{
"invoiceId": "INV-2025-000123",
"status": "ISSUED",
"nextAction": "TRANSMIT"
}
